RF 数学速查表

这就是课程里说的「RF 工具箱」—— 第 3、4 课讲过的所有 dB/dBm/链路预算/SNR 公式和常用值，压缩成一张可回查、可打印的表。需要算 RF 时直接查这里。

① 核心公式

dB（比值，无单位）/ Ratio dB = 10 · log₁₀(P₁ / P₂) dBm（绝对功率，参考 1 mW）/ Absolute power P(dBm) = 10 · log₁₀( P(mW) )　　← 0 dBm = 1 mW 热噪声底（室温 290 K）/ Thermal noise floor N(dBm) = −174 + 10 · log₁₀( B(Hz) ) SNR（信噪比）/ Signal-to-noise ratio SNR(dB) = RSSI(dBm) − N_noise(dBm) 自由空间路径损耗 FSPL（d 单位 km，f 单位 MHz） FSPL(dB) = 20·log₁₀(d) + 20·log₁₀(f) + 32.44

② dB 比值速记（背熟这 8 个）

功率倍数dB逆运算
×1 0 dB 不变
×2 +3 dB −3 dB = ÷2
×4 +6 dB −6 dB = ÷4
×10 +10 dB −10 dB = ÷10
×100 +20 dB −20 dB = ÷100
×1000 +30 dB −30 dB = ÷1000
💡 规律：×10 = +10 dB；×2 ≈ +3 dB。其它值靠组合：×4 = ×2×2 = +3+3 = +6 dB。dB 之间加减 = 功率乘除。

④ 热噪声底（按带宽）

记住 −174 dBm/Hz 这个基准，加上 10·log₁₀(B)。

带宽10·log₁₀(B)噪声底系统
125 kHz +51 dB −123 dBm LoRa 125 kHz
1 MHz +60 dB −114 dBm 蓝牙
20 MHz +73 dB −101 dBm Wi-Fi 20 MHz
40 MHz +76 dB −98 dBm Wi-Fi 40 MHz
80 MHz +79 dB −95 dBm Wi-Fi 80 MHz
160 MHz +82 dB −92 dBm Wi-Fi 160 MHz
💡 带宽 ×10 = 噪声底 +10 dB。这是 Wi-Fi 灵敏度比蓝牙/sub-GHz 差的物理根因，不是接收机技术差。

⑤ SNR → MCS 对照（Wi-Fi）

MCS调制所需 SNR灵敏度（约）
MCS0 BPSK 1/2 ~0 dB −93 dBm
MCS3 16-QAM 1/2 ~9 dB −78 dBm
MCS7 64-QAM 3/4 ~21 dB −64 dBm
MCS9 256-QAM 5/6~29 dB −47 dBm
MCS11 1024-QAM 3/4~36 dB ~−35 dBm
💡 算出 SNR 后查这张表选 MCS。MCS 越高，速率越高，但需更强 SNR。这是自适应速率算法（rate adaptation）的物理依据。

⑥ 完整链路预算流水线（一图记住）

发射功率 − 路径损耗 − 障碍衰减 + 天线增益
 =  RSSI  − 噪声底  =  SNR  →  MCS
💡 所有 RF 计算都是 dB 域加减法。发射、损耗、增益、噪声全用 dB/dBm，像账本一样加减。这就是「链路预算（link budget）」名字的由来。

⑦ 工作算例（参考用）

算例 A：近距离 5 GHz，20 m

+15 (Tx) −66 (FSPL) +3 (天线) = −48 dBm RSSI
−48 − (−101) = +53 dB SNR → MCS11（1024-QAM）

算例 B：隔墙，30 m

+15 (Tx) −70 (FSPL) −12 (墙) +3 (天线) = −64 dBm RSSI
−64 − (−101) = +37 dB SNR → MCS9（256-QAM）

算例 C：边缘场景，弱信号

RSSI = −80 dBm，噪声底 −101 dBm
SNR = −80 − (−101) = +21 dB → MCS7（64-QAM，刚好够）
