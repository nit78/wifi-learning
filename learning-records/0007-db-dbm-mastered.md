# dB/dBm/链路预算已掌握，RF 工具箱第一层成型

**Status**: active

## 学到的

第 3 课（RF 基础重做版）后，学员反馈「算出来了，继续」—— 三道自检题（损耗计算、功率倍数、RSSI→MCS 判断）都能算出来。

**判断：dB/dBm/链路预算这块已掌握。** 这是 RF 数学工具箱的第一层，是后续所有 RF/PHY 课的地基。

具体已掌握的能力（据第 3 课目标）：
- dB 比值换算（+3 dB = ×2，+10 dB = ×10 等）
- dBm 与 mW 的关系（0 dBm = 1 mW，RSSI 为负的原因）
- dB 域加减法 = 功率乘除（链路预算的核心）
- 完整链路预算：发射 - 损耗 + 增益 = RSSI
- RSSI 对照灵敏度表判断 MCS

## 为什么对后续教学重要

1. **RF 工具箱第一层成型**：dB/dBm 运算不再需要从零讲，后续课程可直接使用。这是 LR-0005（RF 被高估）的纠偏成功标志。
2. **第 4 课可按计划推进**：噪声底 + SNR，直接复用第 3 课的 dB 工具（学员刚掌握，趁热巩固）。
3. **完整 RF 链路预算闭环在望**：第 3 课讲了「发射→RSSI」，第 4 课补「RSSI→SNR→MCS」，闭环完成。之后 RF 工具箱成型，可转向协议主线（CSMA/CA）。
4. **信心信号**：学员从「我都不太清楚」（LR-0005）到「算出来了，继续」，节奏感回来了。后续可适度提密度，但仍保持慢节奏铺垫（LR-0005 警示不撤销）。

## Implications

- ✅ <strong>解锁</strong>：第 4 课（热噪声/SNR）按计划推进，直接复用 dB 工具。
- ✅ <strong>解锁</strong>：后续 PHY 课（OFDM 边界、频偏、EVM）可建立在 dB 已熟练之上。
- ⚠️ <strong>保持</strong>：每个新 RF 概念（傅里叶、sinc、复数）首次出现仍需慢节奏铺垫，不因 dB 掌握就放松。

## Evidence

学员原话：「算出来了，继续」。简短、明确、主动要求推进 —— 这是掌握度达标的典型信号（对比上一轮「我都不太清楚，认识应该不足」）。
